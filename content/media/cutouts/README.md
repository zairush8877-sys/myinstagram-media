# cutouts — вырезанные продукты для флэтлэй-коллажей

PNG-вырезки продуктов (в основном на прозрачном/белом фоне), которые собираются
в коллажи-обложки и продуктовые слайды. Например, из этих вырезок собран
`../vitamin-c/dev/01-collage-cover.png`.

## Опись

| Файл | Размер | Что на нём | Тип |
|------|--------|-----------|-----|
| `cut1.png` | 700×1050 | Nécessaire — The Body Lotion (белый тюбик) | вырезка, портрет |
| `cut3.png` | 700×1050 | Erbology — Hemp Seed oil + Matcha + Charcoal (3 предмета столбиком) | вырезка, портрет |
| `cut4.png` | 700×1050 | Curology — cleanser + спрей + rich moisturizer (3 предмета) | вырезка, портрет |
| `cut7.png` | 700×467 | Mender CBD — готовая флэтлэй-раскладка с камнями и зеленью | раскладка, ландшафт |

## Заметки по порядку

- **Дыры в нумерации:** нет `cut2`, `cut5`, `cut6` — это отклонённые/удалённые варианты.
  Нумерацию НЕ пересобираем (движок и коллажи ссылаются на текущие имена); при добавлении
  берём следующий свободный номер.
- **`cut7` — особняк:** это не вырезка на прозрачном фоне, а уже готовая раскладка
  (ландшафт 700×467). Держим здесь же, но помним, что для коллажей она не «строительный
  блок», а самостоятельная картинка.
