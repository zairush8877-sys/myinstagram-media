## Дисклеймер (обязательно, слайд 5)

Класкотерон 5% для лечения **алопеции** — исследуемый препарат, **на момент публикации
не одобрен** FDA/EMA по этому показанию (одобрен только акне-крем Winlevi 1%). Формулировки —
«изучают / фаза 3 / потенциально», без обещаний результата. Не «лечит», а «область науки».

## Факты-первоисточники (для сверки при отрисовке)

- Класкотерон — топический антагонист андрогенных рецепторов; блокирует ДГТ локально
  в фолликуле, без системного антиандрогенного эффекта.
- До него — только миноксидил (1988) и финастерид (1997) с одобрением FDA при АГА.
- Фаза 3 SCALP 1/2 завершена в декабре 2025; прирост плотности до 539% vs плацебо.
- Подача в FDA/EMA — в течение 2026. По показанию «алопеция» пока НЕ одобрен.
