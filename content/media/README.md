# content/media — медиатека контента

Здесь лежат **бинарные ассеты** Instagram-контента (картинки, фоны, вырезки) и манифесты
к ним. Тексты постов, стиль и очередь публикаций — в движковом репо `myinstagram`
(`content/*.md`, `content/queue/*`, `content/visual-style.md`).

> Правило: файлы отсюда переименовывать/двигать **нельзя** без синхронного правок в движке —
> он публикует картинки по этим путям. Наводим порядок через манифесты, а не переименование.

## Два формата контента

- **Сторис** — вертикаль 1080×1920, серия слайдов, живёт в `stories/<тема>/`,
  фоны — в `stories/assets/`. Эталон: `epitalon`.
- **Карусель (пост)** — квадрат/портрет 1080×1350, слайды из Canva, живёт в `<тема>/`
  с `MANIFEST.md` (design_id слайдов + статус). Эталон: `vitamin-c`.

## Соглашения об именах

- Слайды: `NN.png` (сторис) или `NN-<слаг>.png` (карусель), нумерация с `01`.
- Исходные фоны: `sN.jpg` в `stories/assets/`, привязка к слайду — в манифесте темы.
- Вырезки: `cutN.png` (сквозная нумерация по всей медиатеке; дыры = отклонённые/удалённые).
- Черновики и эксперименты — только в подпапке `dev/`, в прод-папку не кладём.
