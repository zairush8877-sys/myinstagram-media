# Референсы стиля (Pinterest Заиры)

> Назначение: держать визуальные референсы В РЕПО, чтобы они не терялись из контекста
> при сжатии истории чата. Описание стиля — в `content/visual-style.md`, история —
> в `content/chat-log.md`.

## Как добавить
Скинь скриншоты сюда файлами и назови по схеме `YYYY-MM-DD-<коротко>.jpg`. Затем
впиши строку в таблицу ниже. (В sandbox-среде Claude бинарники картинок не сохраняются —
их кладёт человек или они приходят через интеграцию.)
